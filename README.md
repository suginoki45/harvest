# Glatch WP starter theme

Starter kit for developing WordPress Theme.

## How to use
### STEP1
```
npm install
```

### STEP2
```
composer install
```

### STEP3
```
npm start
```

## About

Web 制作ユニット[Glatch](https://glatchdesign.com/)が WordPress テーマ制作に使用しているスターターキットです。
